# BxJS-Website

This repository contains code for BxJS Website.  
Primary website content is [BxJS Weekly](https://github.com/BuildingXwithJS/bxjs-weekly) podcast.

## Install with Yarn
* `npm install -g gatsby-cli`
* `yarn global add gatsby-cli`
* `yarn`
* `yarn start`
